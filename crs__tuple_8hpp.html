Detailed Description

Specify sparse matrix as a tuple of CRS arrays.

Example:

Author
Denis Demidov denni.nosp@m.s.de.nosp@m.midov.nosp@m.@gma.nosp@m.il.co.nosp@m.m
// Adapt STL containers:
std::vector<int> ptr;
std::vector<int> col;
std::vector<double> val;
AMG amg( boost::tie(n, ptr, col, val) );
// Adapt raw arrays:
int *ptr;
int *col;
double *val;
AMG amg(boost::make_tuple(n,
boost::make_iterator_range(ptr, ptr + n + 1),
boost::make_iterator_range(col, col + ptr[n]),
boost::make_iterator_range(val, val + ptr[n])
) );
