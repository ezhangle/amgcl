Detailed Description

template<class Matrix>
struct amgcl::backend::viennacl< Matrix >

ViennaCL backend.

This is a backend that uses types defined in the ViennaCL library (http://viennacl.sourceforge.net).

Parameters
MatrixViennaCL matrix to use with the backend. Possible choices are viannacl::compressed_matrix<T>, viennacl::ell_matrix<T>, and viennacl::hyb_matrix<T>.
