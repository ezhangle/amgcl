Enumeration Type Documentation

enum amgcl::gpu_matrix_format

Possible matrix storage formats for GPGPU backends.

Enumerator
GPU_MATRIX_CRS 

Compressed matrix. Fastest construction, slowest operation (on a GPU). Best suited for CPU compute devices.

GPU_MATRIX_ELL 

ELL format. Ideal for matrices with constant number of nonzeros per row on GPU compute devices.

GPU_MATRIX_HYB 

Hybrid ELL format. Best choice for general matrices on GPU compute devices.
