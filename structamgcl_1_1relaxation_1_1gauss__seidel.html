Detailed Description

template<class Backend>
struct amgcl::relaxation::gauss_seidel< Backend >

Gauss-Seidel relaxation.

Note
This is a serial relaxation and is only applicable to backends that support matrix row iteration (e.g. amgcl::backend::builtin or amgcl::backend::eigen).
