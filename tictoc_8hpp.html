Detailed Description

TIC and TOC macros for internal profiling.

Author
Denis Demidov ddemi.nosp@m.dov@.nosp@m.ksu.r.nosp@m.u If AMGCL_PROFILING macro is defined at compilation, then TIC(name) and TOC(name) macros correspond to prof.tic(name) and prof.toc(name). amgcl::prof should be an instance of amgcl::profiler<> defined in a user code similar to:
namespace amgcl { profiler<> prof; }
If AMGCL_PROFILING is undefined, then TIC and TOC are noop macros.
