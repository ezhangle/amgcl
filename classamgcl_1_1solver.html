Algebraic multigrid method.

Parameters
value_tType for matrix entries.
index_tType for matrix indices. Should be signed integral type.
interp_tInterpolation scheme. Possible choices: amgcl::interp::classic and amgcl::interp::aggregation.
level_tClass for storing the hierarchy level structure. Possible choices: amgcl::level::cpu, amgcl::level::vexcl, amgcl::level::ViennaCL<>.

Performs single multigrid cycle.

Is intended to be used as a preconditioner with iterative methods.

The vector types should be compatible with level_t:

  1. Any type with operator[] should work on a CPU.
  2. vex::vector<value_t> should be used with level::vexcl.
  3. viennacl::vector<value_t> should be used with level::ViennaCL.

rhsRight-hand side.
xSolution. Contains an initial approximation on input, and the approximated solution on output.
