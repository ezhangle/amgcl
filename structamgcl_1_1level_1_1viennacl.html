template<gpu_matrix_format Format = GPU_MATRIX_HYB, relax::scheme Relaxation = relax::spai0>
struct amgcl::level::viennacl< Format, Relaxation >

ViennaCL-based AMG hierarchy.

Level of an AMG hierarchy for use with ViennaCL vectors. ViennaCL provides several backends (OpenCL, CUDA, OpenMP) and is thus able to run on various hardware.

Parameters
FormatMatrix storage format to use on each level.
RelaxationRelaxation scheme (smoother) to use inside V-cycles.
